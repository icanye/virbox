ss_user_login API

ss_user_login API

通过 API 登录用户云账号说明

使用场景： 如软件开发商不希望软件用户通过此界面登录账号，可以通过深思提供的 ss_user_login 接口自行实现软件用户的登录

API接口 请在SDK安装目录下 C/dll 目录下查找 ss_user_login.dll（动态库）和 ss_user_login_lib.lib（静态库）

使用说明 用户登录接口详情见：ss_user_login.h API接口声明
